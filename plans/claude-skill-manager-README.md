## Overview

Claude Skill Manager enables easy sharing, installation, and updating of Claude Code skills via git repositories (GitHub, GitLab, etc).

### Features (Planned)

- **Simple CLI**: `claude-skills fetch marimo-notebook [repo]`
- **Git-based**: Uses your existing git authentication
- **Secure**: Configurable allowed sources
- **Version Control**: Track and update skills
- **Multi-platform**: Works with GitHub, GitLab, and self-hosted git
